- **Payment Gateway**: Razorpay integration

- **Auto-Renewal**: Toggle per add-on or plan

- **Cancellation**: Access maintained until end of current billing period

### Migration & Grandfathering
- **Legacy Plan Support**: Existing subscribers can maintain current pricing
- **Migration Notifications**: 30-day advance notice for pricing changes
- **Price Protection**: 12-month price lock for migrated users if new pricing is higher
- **Feature Mapping**: Automatic mapping of old plan features to equivalent add-ons

### Access Control
- **Feature Gating**: Real-time access control based on active subscriptions
- **Entitlement Caching**: 5-minute cache for performance
- **Immediate Activation**: Add-ons activate immediately upon successful payment
- **Immediate Revocation**: Access removed immediately upon expiration (after grace period)

## Organization-Level Subscription Management (B2B)

### Overview

Organization administrators (school_admin, college_admin, university_admin) can purchase subscriptions on behalf of their members (educators and students). This enables centralized billing, bulk purchasing, and streamlined member management.

### Key Features

#### For Organization Admins

**Bulk Purchasing**
- Purchase subscription seats in bulk for educators and/or students
- Volume discounts: 10% off for 50+ seats, 20% off for 100+ seats, 30% off for 500+ seats
- Separate pools for educators and students
- Custom enterprise pricing for 1000+ seats

**License Management**
- Create and manage license pools
- Assign/unassign seats to specific members
- Transfer licenses between members
- Auto-assign licenses to new members
- Real-time seat utilization tracking

**Member Invitations**
- Invite members via email with automatic license assignment
- Bulk invitation support
- Track invitation status (pending, accepted, expired)
- Auto-assign subscriptions upon acceptance

**Billing Dashboard**
- View total subscription costs
- Track seat utilization metrics
- Download invoices for accounting
- Monitor upcoming renewals
- Export payment history

**Add-On Management**
- Purchase add-ons in bulk for member groups
- Assign add-ons to specific educators or students
- Track add-on costs separately from base subscriptions

#### For Organization Members

**Subscription Visibility**
- See features provided by organization
- Distinguish org-provided vs self-purchased features
- View organization admin contact
- Receive expiration notifications
- Option to purchase additional personal add-ons

### Pricing Model

#### Seat-Based Pricing
- Base plan price × number of seats
- Volume discounts applied automatically
- Separate pricing for educator and student seats
- Prorated charges for mid-cycle additions

#### Volume Discount Tiers
| Seats | Discount | Example (₹250/seat) |
|-------|----------|---------------------|
| 1-49 | 0% | ₹250/seat |
| 50-99 | 10% | ₹225/seat |
| 100-499 | 20% | ₹200/seat |
| 500+ | 30% | ₹175/seat |
| 1000+ | Custom | Contact Sales |

### Purchase Flow

1. **Select Plan** - Choose subscription plan and member type
2. **Configure Seats** - Specify number of seats, see volume discounts
3. **Select Members** - Auto-assign to all, select specific members, or create pool
4. **Review & Pay** - Review breakdown, add billing info, complete payment
5. **Assign Licenses** - Assign seats to members from admin dashboard

### License Assignment Options

**Option 1: Immediate Assignment**
- Assign seats to existing members during purchase
- Members get instant access to features

**Option 2: License Pool**
- Create a pool of available seats
- Assign seats later as needed
- Useful for growing organizations

**Option 3: Auto-Assignment**
- Configure rules for automatic assignment
- New members get licenses automatically
- Based on department, grade, or role

### Member Management

**Adding Members**
- Invite via email with auto-license assignment
- Bulk CSV import for large organizations
- Manual assignment from member list

**Removing Members**
- Unassign license to free up seat
- 30-day grace period before permanent removal
- Seat returns to available pool immediately

**Transferring Licenses**
- Move license from one member to another
- Preserves remaining subscription period
- Audit trail maintained

### Billing & Invoicing

**Invoice Details**
- Organization name and GST information
- Itemized breakdown (seats, add-ons, discounts)
- Payment method and transaction ID
- Downloadable PDF format

**Payment Methods**
- Razorpay (credit/debit cards, UPI, net banking)
- Bank transfer for large organizations
- Purchase orders for enterprise customers

**Billing Cycle**
- Monthly or annual billing
- Auto-renewal option
- Prorated charges for mid-cycle changes
- 30-day advance renewal notices

### Subscription Management

**Upgrading**
- Add more seats anytime
- Upgrade to higher plan tier
- Immediate access for new assignments
- Prorated billing

**Downgrading**
- Reduce seat count at renewal
- Switch to lower plan tier
- Scheduled for next billing cycle
- No mid-cycle downgrades

**Renewal**
- Auto-renewal enabled by default
- 30, 15, and 7-day advance notices
- Modify seat count during renewal
- 7-day grace period after expiration

**Cancellation**
- Cancel anytime
- Access maintained until period end
- Seats returned to pool
- No refunds for unused time

### Multi-Tier Organizations

**University → College → Members**
- University admins purchase for all colleges
- Allocate seats to individual colleges
- College admins manage their allocated seats
- Consolidated billing at university level

**Benefits**
- Centralized procurement
- Volume discounts across entire university
- Flexible seat reallocation between colleges
- Unified reporting and analytics

### Access Control

**Organization-Provided Features**
- Managed by organization admin
- Cannot be cancelled by members
- Expires with organization subscription
- Clearly labeled in member view

**Self-Purchased Features**
- Managed by individual members
- Independent of organization subscription
- Can coexist with org-provided features
- Separate billing

### Analytics & Reporting

**For Admins**
- Seat utilization trends
- Member engagement metrics
- Feature adoption rates
- Cost per active user
- ROI calculations
- Exportable reports (CSV/PDF)

**For Members**
- Personal feature usage
- Learning progress
- Skill development tracking
- Certificate achievements

### Support

**Admin Support**
- Dedicated account manager (Enterprise+)
- Priority email support
- Phone support for billing issues
- Implementation assistance
- Training for admin team

**Member Support**
- Standard support channels
- Self-service knowledge base

## Getting Started with Organization Subscriptions

### For New Organizations

1. **Sign up** as school_admin, college_admin, or university_admin
2. **Browse plans** in organization mode
3. **Select plan** and specify seat count
4. **Complete payment** with organization billing details
5. **Invite members** or assign licenses to existing members
6. **Monitor usage** from admin dashboard

### For Existing Organizations

1. **Contact sales** for migration assistance
2. **Review current** individual subscriptions
3. **Calculate savings** with volume discounts
4. **Migrate members** to organization subscription
5. **Consolidate billing** under organization account

### For Members

1. **Receive invitation** from organization admin
2. **Accept invitation** and create account
3. **Access features** provided by organization
4. **View subscription** status in member dashboard
5. **Purchase additional** add-ons if needed
